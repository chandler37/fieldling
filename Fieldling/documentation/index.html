Introduction

This site hosts software tools for doing field linguistics. At present, our goal is to host and disseminate modular components that can be used in larger projects, thus facilitating software reuse and project interoperability.

So far, we have focused primarily on tools for audio and video annotation. Currently, there is detailed documentation for one tool, QuillDriver.
