Introduction

QuillDriver is a tool for creating and linking XML annotations to digital audio and video files. Currently, there are two versions of QuillDriver: a plain vanilla version which works with any kind of data, and a version optimized for Tibetan.

You can try launching either of these versions using Java Web Start by launching on the following links:

  • Generic version (today's build | last stable version)
  • Tibetan version (today's build | last stable version)

Before running QuillDriver, make sure you've installed both Java and QuickTime. Optionally, you can use Java Media Framework instead of QuickTime.
