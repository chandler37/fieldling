Downloads

The following files are currently available for download:

  • Today's API (.zip)
  • Today's source code (.zip)
  • Executable applications, generic flavor:
    • Windows (.zip)
    • OS X (.tar.gz)
  • Executable applications, hacked for Tibetan:
    • Windows (.zip)
    • OS X (.tar.gz)

To run the executable on Windows, you will need to first install (in order):

  1. Java
  2. QuickTime [IMPORTANT! Select 'Custom Install' and tick the 'QuickTime for Java' box.]
Note that you can also use Java Media Framework instead of QuickTime.
